Just a simple farmer application

Requirements:
  • Create a web application for farm management that consists of one page. It will be used only by one farmer.
  • He/she can list, add, and remove animals in the system. Every animal has only a “Name” property that should be unique.
  • The frontend should be written on Angular. The backend is on ASP.NET Core. Use in-memory storage on the server side.
  • The source code should be production ready.
